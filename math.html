Question 01

Let $A=\{x,y,z\}$

(i) Give three elements from the cartesian product $P(A)\times A$

  1. $\left(\{\}, x\right)\in P(A)\times A$
  2. $\left(\{\}, y\right)\in P(A)\times A$
  3. $\left(\{\}, z\right)\in P(A)\times A$

(ii) How many elements are in the cartesian product $P(A)\times A$?

We know for some set, $S$, such that $|S|=n$ the power set of $S$ will have a cardinality of: $|P(S)|=2^{n}$. With this information we can further state that given: $|A|=3\\$We know that:$\\|P(A)|=2^{3}\\|P(A)\times A|=3\cdot2^{3}=24$

A proof for $|A|=n\iff |P(A)=2^{n}|$

We're going to prove this with mathematical induction. Before starting any of the induction work allow us to define a set $S_{n}$ such that for any natural n: $S_{n}=\{a_{1}, a_{2}, a_{3}, \dots, a_{n-1}, a_{n}\}\implies|S_{n}|=n$. This means that for our first inductive step we can state that $S_{0}=\emptyset$ which also means $|S_{0}|=0$. Regarding the powerset of $S$ it's trivial to know the only possible power set is $P\left(S_{0}\right)=\{\emptyset\}$ Further proving our first statement for case $n=0$; $|P(S_{0})|=1=2^{0}\\$Now for the second inductive step we can assume that the following is true: $|P(S_{n})|=2^{n}\\$Our final inductive step is to show that $|P(S_{n+1})|=2^{n+1}$. Now, in order to do this we can state: $S_{n+1}=\{a_{1}, a_{2}, a_{3}, \dots, a_{n-1}, a_{n}, a_{n+1}\}\\S_{n+1}=\{a_{1}, a_{2}, a_{3}, \dots, a_{n-1}, a_{n}\}\cup\{a_{n+1}\}\\S_{n+1}=S_{n}\cup\{a_{n+1}\}\\$By writing $S_{n+1}$ like this it is trivial to state that every subset of $S_n$ is also a subset of $S_{n+1}$. Now we know that there is $2^n$ subsets of $S_{n}$ for $S_{n+1}$ however a given subset either contains $a_{n+1}$ or it does not. So for $S_{n+1}$ there will be $2^n$ subsets that contain $a_{n+1}$ and $2^n$ subsets that do not contain $a_{n+1}$ giving; $|P(S_{n+1})|=2^{n}+2^{n}\\|P(S_{n+1})|=2\cdot2^{n}\\|P(S_{n+1})|=2^{n+1}$ Q.E.D.

Question 03

Let $A$, $B$ and $C$ be sets. Is it true that:

$\left(A\cup B\right)\cap C=A\cup\left(B\cap C\right)$

Prove the statement or find a counter example.

Counter example:

Allow us to define the following: $\\A:=\{x\}\\B:=\{y\}\\C:=\{z\}\\$ With this information we can evaluate both $\left(A\cup B\right)\cap C$ and $A\cup\left(B\cap C\right)$ and compare their outputs. $\left(A\cup B\right)\cap C=\emptyset\\A\cup\left(B\cap C\right)=A\\A\neq\emptyset\\$ Therefore, proving it is false.

Question 04

There is a group of 191 students. 10 of which are taking operating systems, databases and discrete mathematics. 36 are taking operating systems and databases, 20 are taking operating systems and discrete mathematics, 18 are taking discrete mathematics and databases, 65 are taking operating systems, 76 are taking databases and 63 are taking discrete mathematics.

Before starting on any other parts of the question allow us to setup some information more formally and mathematically.$\\|\varepsilon|=191\\O:=\text{Students that study Operating Systems}\\D:=\text{Students that study Databases}\\M:=\text{Students that study Discrete Mathematics}\\$From each of these we can start collecting more information.

10 Students study all three subjects giving us, $|O\cap D\cap M|=10$

36 Students study both operating systems and databases, $|O\cap D|=36$, from which we can further find $|O\cap D\cap\overline{M}|=|O\cap D|-|O\cap D\cap M|=26$

18 Students study both maths and databases, $|M\cap D|=18$, from which we can further find $|M\cap D\cap\overline{O}|=|M\cap D|-|O\cap D\cap M|=8$

20 Students study both maths and operating systems, $|M\cap O|=20$, from which we can further find $|M\cap O\cap\overline{D}|=|M\cap O|-|O\cap D\cap M|=10$

65 Students study operating systems, $|O|=65$, giving, $|O\cap\overline{M}\cap\overline{D}|=|O|-|O\cap M|-|O\cap D|+|O\cap D\cap M|=19$

76 Students study databases: $|D|=76$, giving, $|D\cap\overline{O}\cap\overline{M}|=|D| - |D\cap O| - |D\cap M| + |O\cap D\cap M|=32$

63 Students study mathematics: $|M|=63$, giving, $|M\cap\overline{O}\cap\overline{D}|=|M|-|M\cap O| - |M\cap D| + |O\cap D\cap M|=35$

(i) How many are taking discrete mathematics or operating systems (or both) but not databases?

With the information that we've collected about the system we can now start to calculate other values that might be useful. The one that is being requested in this question is formally written as $|\left(M\cup O\right)\cap\overline{D}|$ from which we can calculate as: $|M| + |O| - |M\cap O| - |M\cap D\cap\overline{O}| - |O\cap D\cap\overline{M}| - |O\cap D\cap M|=64$

(ii) How many are taking none of the three subjects?

In (i) we found that $|\left(M\cup O\right)\cap\overline{D}|=64$ and we're also given that $|\varepsilon|=191$. Now with what we did in the last question we can find out $|M\cup O\cup D|=|\left(M\cup O\right)\cap\overline{D}| + |D| = 140$. Now we can formally write the students that don't study any of the three subjects as $|\overline{M}\cap\overline{O}\cap\overline{D}|$ which is equal to $|\varepsilon|-|M\cup O\cup D|=51$ giving us $|\overline{M}\cap\overline{O}\cap\overline{D}|=51$

Question 05

Let $n>3, n\in\mathbb{N}$ and $A=\{1,2,3,\dots,n\}$.

(i) How many subsets of $A$ contain the elements 1 and 2?

We can approach this question with the same logical reasoning as we did for the proof in question 1.

A power set is a set of all possible subsets of a given set, and we have already proven that the cardinality of a subset, $|P(A)|=2^{n}$. So, using this information we can deduce that $2^{n-1}$ subsets of $A$ contain the element 1 and $2^{n-1}$ subsets do not contain the element of 1. Now for each of these $2^{n-1}$ subsets half of them will contain the element 2 and the other half will not. This means that we can devide our subsets into four groups:

  1. $2^{n-2}$ subsets contain neither 1 or 2.
  2. $2^{n-2}$ subsets contain 1 but not 2.
  3. $2^{n-2}$ subsets contain 2 but not 1.
  4. $2^{n-2}$ subsets contain both 1 and 2.

Which gives us our desired answer, $2^{n-2}$ of the subsets contain both the elements 1 and 2.

(ii) How many subsets B of A have the property that $B\cap\{1, 2\}=\emptyset$?

Using the same ground-work that we applied in (i). It's a trivial statement that in order for $B\cap\{1,2\}=\emptyset$, $B$ must not contain the elements 1 or 2. And we previously deduced that there is $2^{n-2}$ subsets that do not contain 1 or 2.

(iii) How many subsets B of A have the property that $B\cup\{1,2\}=A$?

Now in order for the statement to be true the following must also be true $\{3, 4,\dots,n-1, n\}\subset B$ as $B$ must contain all other elements (With exceptions of 1 and 2 although they may also be present). So this means we're only interested in four of our possible subsets because there's only four possible sets that do/don't contain 1 and 2 but do contain every other element, which is required for $B\cup\{1,2\}=A$.
